<h1 align="center">WazirX Visualizer UI</h1>

<div align="center">

A front end visualizer for WazirX Data Collector.


## Usage

1. Clone project code.

```bash
git clone https://github.com/adityak74/wazirx-visualizer-ui.git my-project
cd my-project
```

2. Installation dependence.

```bash
yarn install
```

Or

```bash
npm install
```

3. Start local server.

```bash
npm run start
```

4. After the startup is complete, open a browser and visit [http://localhost:7000](http://localhost:7000), If you need to change the startup port, you can configure it in the `.env` file.
